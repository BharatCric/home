— Why we built it

Every weekend, a million matches are played across India. Tape-ball thrillers in tight gullies, weekend leagues on dusty grounds, college friendlies that end in chai. Almost none of them are scored properly.

Existing apps are bloated, slow, and built for stadiums — not streets. We wanted something different. Something you can pick up mid-over and just start scoring. No accounts, no friction, no learning curve.

BharatCric is that app. A clean, fast, beautifully made cricket scoring companion that respects your time and treats every backyard match like it matters. Because it does.

Questions, briefly.

— Six things people ask
Is BharatCric free?
Yes. Quick Scoring is free forever. Profile matches are free during beta, and we'll keep the core free at launch. We'll only charge for advanced team features down the line — if ever.
Do I need an account to score?
No. Quick Scoring works anonymously. You only need an account if you want stats to be tracked to your profile, or if you want to host profile matches with real players.
Can multiple people score the same match?
Yes. The match host can add up to 3 co-hosts. Anyone on that list can score the next ball — whoever's free, takes the phone.
Does it work offline?
Yes. Score offline at the ground; everything syncs when data returns. You can also share live match links so friends watch from anywhere.
iOS? Web?
Android only for now. iOS is on the roadmap. The match-viewer link works on any browser — share a match URL and anyone can watch live.
Where can I read the privacy policy?
Right here: bharatcric.github.io/privacy-policy. Plain language. No surprises.
